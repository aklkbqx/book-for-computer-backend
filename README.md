## 🚀 Installation and Usage

### Installation

You can use either Bun or npm to install and run this project

```bash
# Using Bun
bun install

# Or using npm
npm install
```

### Running the Project

```bash
# Using Bun
bun run dev

# Or using npm
npm run dev
```
